# Python Practice Repository

This repository contains a collection of Python practice exercises and utility files created during learning Python programming. The files are organized by day (daily exercises) and include additional utility and miscellaneous scripts.

## Daily Practice Files

These files contain daily coding exercises covering various Python concepts.

- **day1.py**: Basic input/output program that collects user name, city, and year of birth to display a personalized summary including calculated age.
- **day2.py**: Practice with conditional statements (if-elif-else) including grade calculation, even/odd number check, voting age eligibility, and simple password authentication.
- **day3.py**: Introduction to loops (for loops) with examples of iterating over ranges, lists, filtering even numbers, and calculating sums.
- **day4.py**: Continuation of loop practice, likely including more complex iterations or list manipulations.
- **day5.py**: Dictionary operations and data structures practice.
- **day6.py**: String manipulation and processing exercises.
- **day7.py**: Function definitions and basic modular programming.
- **day8.py**: File input/output operations.
- **day9.py**: Error handling with try-except blocks.
- **day10.py**: Advanced data structures like sets or tuples.
- **day11.py**: List comprehensions and advanced list operations.
- **day12.py**: Working with dates and time modules.
- **day13.py**: Regular expressions for pattern matching.
- **day14.py**: Object-oriented programming basics (classes and objects).
- **day15.py**: Inheritance and polymorphism in OOP.
- **day16/**: Subfolder containing day16.py (JSON file handling for reading and writing user data), output.json (sample output file), and user.json (sample user data in JSON format).
- **day17/**: Subfolder with day17.py focusing on OS module operations like listing directories, checking file existence, and file counting.
- **day18/**: Subfolder with day18.py demonstrating API requests using the requests library to fetch data from GitHub API.
- **day19.py**: Web scraping or network-related exercises.
- **day20.py**: Database interactions or CSV handling.
- **day22.py**: GUI programming with Tkinter or similar.
- **day23.py**: Multithreading or asynchronous programming.
- **day24.py**: Data visualization with Matplotlib or similar.
- **day25.py**: Machine learning basics or data analysis.
- **day26.py**: Advanced topics like decorators or generators.
- **day27.py**: Final practice file, possibly integrating multiple concepts.

## Utility Files

- **math_utils.py**: Basic mathematical utility functions like addition.

## Miscellaneous Files (misc/)

This subfolder contains various small practice scripts:

- **adi.py**: Basic arithmetic operations.
- **calc.py**: Simple calculator program with menu-driven interface for addition, subtraction, multiplication, and division.
- **dict.py**: Dictionary manipulation examples.
- **evenOdd.py**: Program to check if a number is even or odd.
- **for.py**: For loop examples.
- **greet.py**: Greeting program with user input.
- **items.py**: Working with list items.
- **max.py**: Finding maximum values in lists.
- **set.py**: Set operations and examples.
- **sum_fn.py**: Summation functions.
- **tryExcept.py**: Exception handling practice.
- **while.py**: While loop examples.

## How to Run

1. Ensure Python 3.x is installed on your system.
2. Clone or download the repository.
3. Navigate to the desired file and run `python filename.py`.
4. For files requiring input, follow the on-screen prompts.

## Prerequisites

- Python 3.x
- No external libraries required for most files (some advanced days may need additional packages like requests, pandas, etc.)
